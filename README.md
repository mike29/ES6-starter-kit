## ES6 a simple starter kit
Simple starter kit for es6 web application  ES6, HTML, CSS, Gulp, babel, express on nodeJS.
 > The starter kit can be used for a simple project or practice purpose.<br>
 Start with
    ```
 npm install
    ```
 Start command
    ```
  gulp serve

    ```
  Build
    ```
    gulp serve:dist

    ```
